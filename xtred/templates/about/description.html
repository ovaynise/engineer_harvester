О проекте ©️Engineer

(Beta)

©️Engineer — это многофункциональная платформа для управления инженерными проектами. Она включает базу данных для отслеживания проектов, подрядчиков и исполнителей, а также функционал для создания отчетов и управления гарантийными обязательствами.

Основные функции

  • Управление проектами
  • База данных подрядчиков и исполнителей
  • Формирование отчетов
  • Управление гарантийными обязательствами
  • REST API для интеграции с внешними сервисами

Асинхронный Telegram-бот

Асинхронный Telegram-бот с ИИ работающий в связке с данным сайтом, с возможностью установки напоминаний, информирования о погоде и управления уровнями доступа для администратора.

Использованные технологии

  • Backend: Python, Django REST Framework, PostgreSQL
  • Фронтенд: на Django: CSS, HTML, Bootstrap 5
  • Асинхронная работа с ботом: Aiogram
  • Веб-сервер: Gunicorn, Nginx
  • Аутентификация и авторизация: DJOSER
  • Контейнеризация: Docker, Docker-compose
  • CI/CD: GitHub Actions

Автоматизация деплоя

Проект использует GitHub Actions для CI/CD. Автоматическое развёртывание происходит при каждом пуше в ветку main.
